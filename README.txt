でんしゃでゴー！

内容:
5歳くらい向けの、電車と数字のクイズWebアプリです。
Safariで開いてホーム画面に追加すると、iPhoneでアプリのように使えます。

使い方:
1. index.html をブラウザで開く
2. 「はじめる」を押す
3. 数字を選ぶ

iPhoneで使う方法:
1. ファイル一式を任意のWebサーバーに置く
   例: GitHub Pages, Vercel, Netlify など
2. iPhoneのSafariでURLを開く
3. 共有メニューから「ホーム画面に追加」を選ぶ

カスタマイズ:
- index.html 内の questions 配列を書き換えると問題を追加できます
- 絵文字の電車を画像に置き換えることもできます
- 小田急線モチーフの駅名や色に変更できます
